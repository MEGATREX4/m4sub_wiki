**Капуста** — це кастомна сільськогосподарська культура, додана в мод M4Terraform. Вона призначена для вирощування на грядках і дає стабільне джерело їжі. Завдяки спеціальній системі на базі Polymer, культура працює як на сервері, так і відображається коректно для гравців без встановленого клієнтського модa.

На відміну від ванільних культур, капуста має зручну механіку **швидкого збору** правою кнопкою миші, що значно прискорює масове фермерство.

## Технічні характеристики

| Параметр                    | Значення                          |
|----------------------------|-----------------------------------|
| Внутрішній ID блоку        | `m4terraform:cabbage_crop`        |
| ID насіння                 | `m4terraform:cabbage_seeds`       |
| ID продукту                | `m4terraform:cabbage`             |
| Максимальна стадія росту   | 7                                 |
| Базовий шанс росту         | 15% за random tick                |
| Врожайність (зріла)        | 2–4 капусти                       |
| Дроп насіння               | 50% шанс при швидкому зборі      |
| Потреба на посадку         | Зорана земля (Farmland)           |
| Сумісність                 | Bone Meal, автоматичний полив     |

## Вирощування

Щоб виростити капусту, гравець повинен спочатку отримати **Насіння капусти** (`cabbage_seeds`).

### Умови вирощування:
- Садити можна **тільки на зораній землі** (Farmland).
- Культура має **7 стадій росту** (від 0 до 7).
- Для росту потрібен рівень освітлення **8 і вище**.
- Швидкість росту залежить від вологості ґрунту (moisture). Вологий farmland значно прискорює процес.
- Базовий шанс зростання — **15%** за випадковий тік (random tick).

Капуста повністю інтегрована в універсальну систему `CropManager`, яка використовується для всіх кастомних культур мода.

## Механіка росту та взаємодії

Культура використовує складну систему на базі **Polymer**:

- Блок фізично є `PolymerCropBlock`, який наслідує `PolymerTexturedBlock` та `BlockWithElementHolder`.
- Візуально культура відображається через **Item Display Entity** з кастомними моделями (`cabbage_crop_stage_0` … `cabbage_crop_stage_7`).
- Всі культури мода використовують один спільний `PLANT_BLOCK` стан з прозорою текстурою, щоб не займати зайві blockstate-с лоти.

### Взаємодія з гравцем:

- **Ліва кнопка миші (ЛКМ)** — звичайне руйнування блоку.
- **Права кнопка миші (ПКМ)** — **швидкий збір**. Якщо капуста зріла (стадія 7), при натисканні ПКМ:
  - Випадає врожай гравцеві в руки.
  - Кущ автоматично скидається до 0 стадії (без потреби пересаджувати).
  - Це значно зручніше для великих ферм.

- **Кісткове борошно (Bone Meal)** повністю підтримується. При використанні капуста отримує від 2 до 5 стадій росту за раз.

## Збір врожаю та дроп

### При зборі зрілої капусти (стадія 7):

- **Основний дроп:** від **2 до 4** одиниць капусти (`cabbage`).
- **Додатковий дроп:** 50% шанс отримати 1 насіння капусти.

### При руйнуванні недозрілої капусти:

- Випадає **лише 1 насіння** капусти.

### При швидкому зборі (ПКМ):

- Врожай випадає відразу в інвентар.
- Кущ залишається на грядці на стадії 0.
