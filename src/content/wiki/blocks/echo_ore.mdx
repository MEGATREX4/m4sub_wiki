# Руда відлуння

**Руда відлуння** — цінна руда, яка знаходиться у Прірві МОроку. При видобутку дає **Блок Ехо Руди** — основну сировину для крафту нових предметів.

## Розташування

- **Біом:** будь-який біом Прірви мороку
- **Висота:** від -90 до 0 висоти
- **Рідкість:** Рідкісна

## Механіка дропу

- При вибуху близько вибуху — частина дропу втрачається (функція `explosion_decay`)
- Дає сам блок для переміщення якщо просто добути
- Можна розмістити в іншому місці та видобути знову

## Використання

**Блок Ехо Руди використовується для:**
- Декорування магічних локацій
- Крафту спеціальних компонентів

## Властивості блоку

- **Твердість:** Середня (як в глибосланцю)
- **Звуки:** Звук дилатації
- **Частинки при розбитті:** глибосланцеві частинки
